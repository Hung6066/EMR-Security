 info Hướng dẫn sử dụng

Cách sử dụng API Key:

  1. Tạo API key mới với tên mô tả
  2. Sao chép API key (chỉ hiển thị một lần)
  3. Thêm header vào request: X-API-Key: your-api-key
  4. Gọi API endpoint bình thường

Ví dụ:

curl -H "X-API-Key: emr_abc123..." http://localhost:5000/api/patients

Lưu ý bảo mật:

  • Không chia sẻ API key với người khác
  • Lưu trữ API key ở nơi an toàn
  • Thu hồi ngay nếu nghi ngờ bị lộ
  • Sử dụng IP whitelist nếu có thể
  • Đặt thời hạn hợp lý cho API key
